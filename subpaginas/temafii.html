Antecedentes y perspectivas históricas.

# El comienzo de todo (antes de 1940) - Alan Turing considerado uno de los padres de la informática y la inteligencia artificial: - Test de Turing: test para evaluar el nivel de inteligencia de una máquina en función de sus respuestas, indistinguibles del ser humano. - Concepto de máquina de Turing: autómata que reconoce cualquier lenguaje formal - La bomba / Phoenix: computadora que buscaba la configuración de los rotores de la máquina enigma con una cadena de deducciones lógicas. --- # Primera generación (40-58) - Válvula de vacío: componente electrónico utilizado para amplificar conmutar o modificar una señal eléctrica - La memoria se construyó usando tubos de Mercurio liquido. - El almacenamiento interno estaba compuesto por un tambor que giraba rápidamente. - Los programas se cargaban a través de tarjetas perforadas. - ENIAC: de los primeros ordenadores de propósito general, según la programación servía para una tarea u otra, utilizaba tarjetas perforadas. - La inteligencia artificial surgió para imitar el razonamiento y comportamiento humano, Conferencia Dartmouth (56) - “El perceptron” (57): Frank Rosenblatt trata de imitar lo que hacen las neuronas con la primera red artificial de neuronas. --- # Segunda generación (59-64) - El transistor: produce un señal de salida e- en respuesta a una señal de entrada que puede ser amplificada, oscilará, conmutada y rectificada. - Puede funcionar como interruptor permitiendo El Paso de corriente - Puede funcionar… - Computadoras centrales: dispositivos centralizados, para BIG DATA, planificación de recursos empresariales y las transacciones (grandes). - Sistemas de tiempo compartido - Terminales simples - Interfaces no amigables - Podían programarse - COBOL (59): primer lenguaje universal de programación de alto nivel, creado para ser ejecutado en cualquier computadora mediante compilador. Grace Hopper mayores precursores nivel privado y militar EEUU. --- # Tercera generación (64-71) - Aparecen los circuitos integrados, chips o microchips: - realizan una función específica - se construyen usando materiales semiconductores - Tipos: - Analógicos: funcionan como amplificadores u osciladores de la señal - Digitales: funcionan como puertas lógicas, se trabaja con valores binarios - Ley de Moore: dice que cada 2 años de duplicara el número de transistores en un chips. - Aparecen lenguajes como BASIC o Pascal, creados para ser lenguajes de enseñanza para nuevos programadores. - Creación de ELIZA, primer robot conversacional. - Creación de Shakey, robot capaz de razonar utilizando el GPS. - Creación del algoritmo A*, para la resolución de problemas de búsqueda, optimización y razonamiento. --- # Cuarta generación (71-81) - Microprocesador: ejecuta instrucciones de lenguaje binario, como operaciones aritméticas y lógicas simples. Tiene dos elementos básicos: - Arithmetic Lofic Uni (ALU): circuito digital que permite ejecutar operaciones aritméticas y lógicas entre valores almacenados. - Banco de registros: registros de memoria de pequeña capacidad y alta velocidad. - Primer microprocesador: INTEL 4040 en el 71, con 4 bits - Creación de redes informáticas de alta velocidad - Definición: plataforma para el intercambio de datos y recursos compartidos entre diferentes sistemas. - Tipos:- LAN: red de área local - MAN: red “metropolitana”, permite conectar las LAN y WLAN entre si. - WAN: red que permite conectar todas entre si, utilizada por proveedores de internet - WLAN: red de área local inalámbrica. - UNIX: familia de sistemas operativos multitarea y multiusuario. - basado en nodos - comunicación de procesos mediante “tuberías” - para tareas complejas y flujos de trabajo - C: lenguaje estructurado de propósito general, considerado de bajo nivel, muy utilizado para la construcción de sistemas operativos --- # Quinta generación (81-95) - Aparece la computadora personal: una máquina programable, procesa datos de entrada, que generan información que se envía a las unidades de salida. - Aparece internet: conjunto descentralizado de redes interconectadas, que componen una única red lógica con alcance global. - aparecen las páginas web - aparecen nuevos servicios: comercio, correo electrónico- Aparece el lenguaje HTML: lenguaje definido para páginas web mediante el uso de etiquetas. - Aparece Python: hecho para que el código sea más fácil de leer para los programadores. Muy utilizado para Big Data y IA --- # Sexta generación (95-actualidad) - Empiezan los primeros Smartphones - (IBM presenta Deep Blue) - Se crea facebook en 2004 - Comienza la era de los robots asistentes - DARPA :concurso de vehículos autónomos - Primeras redes neuronales profundas eficientes para la identificación de objetos mediante Artificial Visión. - El desarrollo de aplicaciones se convierte en una industria muy grande - Vehículos autónomos - Robots que usan IA para interactuar y razonar - Primeros ordenadores cuánticos que utilizan Qubits en vez de bits.
